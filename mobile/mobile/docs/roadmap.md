# Roadmap

## Phase 1 — Product skeleton
- Set up monorepo
- Build mobile shell with polished UI library
- Build backend with stable mock endpoints
- Document architecture and deployment

## Phase 2 — Connected demo
- Connect mobile app to backend
- Create real lesson loading
- Create practice session flow
- Upload audio from device
- Return result payload to mobile

## Phase 3 — Pronunciation engine v1
- Integrate text-to-phoneme mapping
- Add forced alignment
- Add heuristic scoring
- Generate targeted coaching tips

## Phase 4 — Pronunciation engine v2
- Improve phoneme confidence logic
- Add stress and rhythm analysis
- Track weak sounds over time
- Improve personalized feedback

## Phase 5 — Production readiness
- Add real auth
- Add persistence for users and results
- Add analytics and crash reporting
- Add CI/CD and monitoring
- Prepare Android/iOS release pipeline
